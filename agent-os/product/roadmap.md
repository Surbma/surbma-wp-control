# Product Roadmap

## Phase 1: MVP

Core features for the first complete release of the WordPress Optimization toolkit:

- **Site health / status dashboard** — at-a-glance overview of key WordPress installation metrics and settings
- **Image & media management** — image size overview, EXIF-based rotation fix, per-site usage scanner (library and content)
- **External link checker** — scan posts and pages for broken external links

## Phase 2: Post-Launch

Features planned for future releases:

- **Performance optimization checks** — caching status, page speed indicators, database size, autoload options analysis
- **Security status checks** — file permissions, debug mode detection, outdated PHP version warnings, and other common security issues
